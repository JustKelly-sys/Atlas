## What Atlas is (context)
Atlas is a payroll operations suite for mid-market companies across six countries. Ships five AI automations covering specific EOR frictions. ADP launched an enterprise Payroll Variance AI Agent on 28 January 2026 — Atlas is the mid-market equivalent, uncontested.

## What this page is
This is `/app/payroll/variance` — Variance Narrator (NP-06). Every cycle variance gets a plain-language explanation generated by Gemini 2.5 Flash, grounded in the actual line items. The page also exposes the MCP server so a Finance lead can query variance data conversationally from Claude Desktop.

## Functional requirements
- Reads from Supabase `variances`, `payroll_cycles`, `countries`, `payroll_line_items`.
- Each variance row has: `variance_amount`, `variance_pct`, `threshold_crossed`, `cause_category` (one of `headcount` / `rate` / `fx` / `statutory` / `bonus` / `unexplained`), `narration_text` (nullable), `narration_model`.
- "Narrate" button per un-narrated row hits `/api/variance/narrate/[id]` → Python variance-narrator-mcp service on Render → Gemini 2.5 Flash with the variance + prior-cycle line items in the prompt → returns narration paragraph + cause classification → writes back.
- "Ask in Claude" opens a right-side drawer with a natural-language input. This is a stub in-product (shows what it would do with canned responses) but the real MCP server is live — the drawer's footer shows the MCP config snippet to add to Claude Desktop.
